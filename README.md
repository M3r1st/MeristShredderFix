# [WOTC] Shredder Fixes

Steam Workshop: https://steamcommunity.com/sharedfiles/filedetails/?id=3707570556

## What is this?
This mod fixes the following long-standing issues with how Shredder works:
1. Pistols used as primary weapons do not benefit from Shredder.
2. Bonus shred amount from Shredder is not scaling properly with non-standard weapon tech levels.

## Configuration
This mod has an extensive configuration that can be found in `XComShredderFixes.ini`.

## Compatibility
* Compatible with **Long War of the Chosen**.
* **NOT** compatible with other mods that attempt to fix the above-mentioned issues. The only exception to this is **Weapon Fixes** by *Mr. Nice*. However, most of the fixes from this mod have been included in the Highlander, making it, effectively, obsolete.
